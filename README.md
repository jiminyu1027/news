## 사용 기술

- HTML
- CSS
- JavaScript
- Bootstrap

## 기능

- 유저는 최신 뉴스를 볼 수 있다.
- 뉴스에는 사진, 제목, 내용, 출처, 날짜가 있다.
- 뉴스를 sport, tech, business등 카테고리별로 볼 수 있다.
- 뉴스를 검색할 수 있다.
- 웹사이트는 반응형 웹페이지다.
- pagenation으로 다음 뉴스정보를 받아볼 수 있다.
